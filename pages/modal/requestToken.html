The easiest way to connect your server to MCGameInfo is by installing the plugin for Bungeecord or Bukkit.
Update its config.yml file with the following information:

# You can get the required ID and Token by adding your server at https://mcgame.info/account#servers
server:

  # Unique ID for your server
  id: "{{ serverId }}"

  # Request Token for your server
  token: "{{ requestToken }}"

If you're not interested in writing any code, simply enable gameDetector.enabled in the config and adjust the settings. The plugin will automatically update the game based on server/world names.

Otherwise, you can call the following API methods: 

// Bukkit/Spigot
GameInfoClient client =
		((org.inventivetalent.mcgameinfo.bukkit.Plugin) Bukkit.getPluginManager().getPlugin("MCGameInfo")).getClient();

// Bungeecord
GameInfoClient client =
		((org.inventivetalent.mcgameinfo.bungee.Plugin) ProxyServer.getInstance().getPluginManager().getPlugin("MCGameInfo")).getClient();


// Join Methods
client.joinServer(String username, UUID uuid);

client.joinGame(String username, UUID uuid, String gameName);

// Leave Methods
client.leaveGame(String username, UUID uuid);

client.leaveServer(String username, UUID uuid);
