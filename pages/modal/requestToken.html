The easiest way to connect your server to MCGameInfo is by installing the plugin for Bungeecord or Bukkit.
Update its config.yml file with the following information:

# You can get the required ID and Token by adding your server at https://mcgame.info/account#servers
server:

  # Unique ID for your server
  id: "{{ serverId }}"

  # Request Token for your server
  token: "{{ requestToken }}"
